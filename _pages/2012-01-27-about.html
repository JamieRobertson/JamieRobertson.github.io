I am an enthusiastic web designer and front-end developer. I work at Monki in Stockholm and I like to make stuff with HTML5, CSS and JavaScript.

About this site

This site was originally called Jamie Robertson Innit but was deemed to be too facetious. I remade everything in late 2013. The portfolio area slider below is built from scratch with some simple jQuery functions. It uses hardware accelerated css transforms and transitions, and on older browsers it falls back to using css positioning and transitions, and on even older browsers it uses regular jQuery animations. I also try to keep the slider in its last position by using a session cookie.

The icons in the navigation area are animated with css keyframe animations. If the browser doesn't support css keyframe animations, it will fallback to using the Spritely library to display animated javascript sprites.

If you like anything on my site please feel free to take a look at the source code or ask me a question about it.

My favourite resources

My most frequented resource is of course Stack Overflow due to its overwhelming wealth of information. I am also a keen follower of blogs like A List Apart, CSS Tricks, and most recently, the Codrops blog by Tympanus.

Typography

The headers on this site are set in Lato, which is available from Google web fonts. I use it in three weights and also make use of letter spacing and border-bottoms (not text-decoration underlines) to sex it up.
