This website was built for an artist who wanted a very organized portfolio site that was easy to update. I decided to use Wordpress as a CMS but heavily modified it, changing the admin interface by exploiting the featured image functionality so that it was image-centric, as opposed to the default Wordpress interface, which focusses on text. The result is an admin area perfectly suited to this particular role.
